# Raices de Amor

Invitacion digital interactiva para celebrar 25 anos de matrimonio y 25 anos de ministerio de Zoraida y Alfonso.

## Como publicar en Netlify

1. Entra a Netlify.
2. Elige **Add new site**.
3. Usa **Deploy manually**.
4. Arrastra la carpeta completa del proyecto.
5. Netlify generara un enlace publico para compartir.

El proyecto es estatico: solo usa HTML, CSS y JavaScript. No necesita XAMPP, PHP, MySQL ni Firebase.
